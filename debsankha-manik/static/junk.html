Junk

Here I list a few useful scripts and programs I wrote: some for simulation-based projects, others just for fun. I am afraid that some of them might not be sufficiently commented, since I naively assumed that nobody except me would ever read them. However, you are welcome to contact me if you need any assistance regarding them.
  • Lyrical: A lyric fetcher for Rhythmbox/Amarok hide
    This program fetches the lyrics of the song currently playing in Rhythmbox or Amarok and displays them in a GTK window, a la lyricscripts of amarok. This is different, however, from all of them in one respect: instead of searching in some specific lyric database, it simply googles for the lyric, parses the HTML and extracts the lyrics. As a result, this can return the lyric of almost any song you throw at it. For more details, visit the project homepage or checkout the code from the github repository.
  • A simulation of 2-d Ising spin system using VPython expand
    I wrote this program for my summer project in 2009. VPython is used to visualize the system as it evolves from a randomly assigned initial spin configuration according to the Metropolis algorithm. The ribbon like structures are formed for t<1. Requires python-visual package on Ubuntu or Debian.
  • The same ising model simulation in C using ncurses expand
    I had to rewrite the same program in C because the python-visual package was broken in Ubuntu 9.10 (Karmic Koala). The visualization part was done using ncurses, mostly copied from the code by abhidg. Requires libncurses-dev package for compilation on Ubuntu or Debian.
  • A simulation of predator-prey system expand
    Written for my Spring 2010 biology project, this program exhibits coupled oscillation in both predator and prey population. Requires pygtk and cairo.
  • Script to sync a music library with Nokia smartphones expand
    This tiny python script uses the mighty rsync to get the job done. I wrote this after I made a disastrous attempt to sync my Nokia 5800 XpressMusic with iTunes : the official plugin from Nokia decided to delete all the existing files in my phone without even asking me!
